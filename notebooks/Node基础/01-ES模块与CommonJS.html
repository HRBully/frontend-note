    ES模块

    # ES模块

    # 使用

    在 Node.js 的配置文件package.json中设置参数type: module。

    # 语法

    导出

    export { 模块的公共API }
// 或者
export default 单个API

    导入

    import { API名字 } from 模块名

    # 实例

     // utils.js
const fn = (name) => {
    console.log("我的名字叫做"+(name || "猪痞恶霸"))
}
// 导出
export {
	fn
}

    // index.js
// 引入
import { fn } from './index.js'

fn() // 我的名字叫做猪痞恶霸
fn("ned") // 我的名字叫做ned

    # 注意

    • 通过export导出函数或者变量，可以导出多个
    • 通过export default 单个对象只能导出单个API
