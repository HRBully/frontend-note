## 重复的DNA

力扣链接：[187. 重复的DNA序列 - 力扣（LeetCode）](https://leetcode.cn/problems/repeated-dna-sequences/)

```js
var findRepeatedDnaSequences = function(s) {
    let res = new Set()
    let dnaSet = new Set()
    if(s.length <= 10) {
        return [...res]
    }
    let start = 0, end = 10
    while(end <= s.length) {
        let cur = s.slice(start,end)
        if(dnaSet.has(cur)) {
            res.add(cur)
        }else {
            dnaSet.add(cur)
        }
        start++
        end++
    }
    return [...res]
};
```

- 滑动窗口分割字符串
- 双哈希结构存储结果与是否存在标识
